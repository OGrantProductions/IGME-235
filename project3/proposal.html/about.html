Minesweeper

High Concept

This game is Minesweeper but the player can customize the board size and the number of bombs on the board.

Genre/Platform/Story/Aesthetics

Minesweeper is a strategy, puzzle game. This version will be a desktop web game with a more modern look and nicer color palette than the original Minesweeper. This game will have as much story as the original minesweeper, which is to say, none at all.

Gameplay

In the setup of the game, the player will be able to select how many rows and columns they want for their board, as well as how many bombs will be on the board. During gameplay, everything will be done using the mouse. The player will left-click to reveal a tile and right-click to flag a tile as a mine. They will only have as many flags as there are bombs on the board. The game ends when the player either left-clicks a mine, or properly reveals all of the tiles on the board./p>
